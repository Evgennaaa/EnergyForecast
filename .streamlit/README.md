# Shadcn-ui Inspired Design

Приложение использует дизайн, вдохновлённый библиотекой компонентов shadcn-ui.

## Особенности дизайна

### Цветовая схема
- **Фон**: Тёмный slate-900 (#0F172A)
- **Карточки**: Slate-800 (#1E293B)
- **Акцент**: Sky blue (#0EA5E9)
- **Текст**: Светлый slate-100 (#F1F5F9)

### Компоненты
- Скруглённые углы (border-radius: 0.5rem)
- Тени для глубины
- Плавные переходы и анимации
- Современная типографика (Inter font)
- Минималистичный дизайн

## Файлы конфигурации

- `.streamlit/config.toml` - настройки темы Streamlit
- `.streamlit/style.css` - кастомные CSS стили

## Запуск

Приложение автоматически применяет стили при запуске:

```bash
streamlit run app.py
```

## Кастомизация

Для изменения цветов отредактируйте переменные CSS в `.streamlit/style.css`:

```css
:root {
    --primary: 14 165 233; /* Sky blue */
    --background: 15 23 42; /* Slate 900 */
    /* ... */
}
```
