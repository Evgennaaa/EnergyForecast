# Эксперименты

Эта папка предназначена для экспериментов и дополнительных исследований.

## Назначение

Здесь можно размещать:
- Экспериментальные ноутбуки с новыми подходами
- Тестирование альтернативных моделей
- Исследование новых признаков
- Валидация гипотез
- Временные эксперименты перед интеграцией в основной пайплайн

## Структура

Рекомендуется создавать подпапки для разных типов экспериментов:
- `feature_engineering/` - эксперименты с признаками
- `model_alternatives/` - альтернативные модели
- `hyperparameter_tuning/` - эксперименты с гиперпараметрами
- `data_exploration/` - дополнительный анализ данных

## Примечание

Экспериментальные ноутбуки не являются частью основного пайплайна и могут содержать нестабильный или незавершенный код.
