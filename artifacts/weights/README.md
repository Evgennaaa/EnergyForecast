# Модель XGBoost

Поместите вашу обученную модель XGBoost в этот каталог с именем `xgboost_model.pkl`.

## Формат модели

Модель должна быть сохранена с использованием `joblib`:

```python
import joblib
import xgboost as xgb

# После обучения модели
model = xgb.XGBRegressor(...)
model.fit(X_train, y_train)

# Сохранение
joblib.dump(model, 'artifacts/weights/xgboost_model.pkl')
```

## Требования к модели

Модель должна быть обучена на данных с следующими признаками:
- `hour` - час дня (0-23)
- `day_of_week` - день недели (0-6)
- `month` - месяц (1-12)
- `day_of_year` - день года (1-365)
- `is_weekend` - выходной день (0 или 1)
- `lag_1h` - значение 1 час назад
- `lag_24h` - значение 24 часа назад
- `lag_168h` - значение 168 часов назад
- `rolling_mean_24h` - скользящее среднее за 24 часа
- `rolling_mean_168h` - скользящее среднее за 168 часов
- `rolling_std_24h` - скользящее стандартное отклонение за 24 часа

Модель должна быть регрессором XGBoost, который предсказывает энергопотребление в кВт·ч.
